### Active Projects

 - Integrating the [Bouncy Castle crypto provider library](https://www.bouncycastle.org/) into [JCrypTool](https://github.com/organizations/jcryptool/). The core of it is built with Scala and uses a Domain Specific Language to provide generic GUI and console functionality. The long-term plan is to base more and more of the core implementation on the [Shapeless](https://github.com/milessabin/shapeless) and functional lenses and weave it into the [Ammonite](http://ammonite.io) REPL. Far goal is to utilize Lightweight modular staging to give it the potential to be viable in performance-critical settings. As a modelling stretch-goal, I plan to ultimatively make the DSL use [Free Applicatives](https://en.wikipedia.org/wiki/Applicative_programming_language) (from category theory) to base it on firm theoretical ground.

 - Making an acoustic guitar into a human interface device using an arduino. [Project log with pictures on github](http://simlei.github.io/VAR2017Project)
