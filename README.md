### Active Projects

 - Integrating the [Bouncy Castle crypto provider library](https://www.bouncycastle.org/) into [JCrypTool](https://github.com/organizations/jcryptool/). The core of it is built with Scala and uses a Domain Specific Language to provide generic GUI and console functionality that is configurable and extensible via OSGi. The long-term plan is to base more and more of the core implementation on the [Shapeless](https://github.com/milessabin/shapeless) and [Scalaz](https://github.com/scalaz/scalaz) libraries and ultimatively make the DSL use [Free Applicatives](https://en.wikipedia.org/wiki/Applicative_programming_language) (from category theory) to base it on firm theoretical ground.

 - Making an acoustic guitar into a human interface device using an arduino. [Project log with pictures on github](http://simlei.github.io/VAR2017Project)
